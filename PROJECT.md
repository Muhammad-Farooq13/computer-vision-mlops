## 📦 What's Inside?

A production-ready computer vision project featuring:
- 🤖 Multiple CNN architectures (ResNet, EfficientNet, VGG)
- 🚀 Flask REST API for deployment
- 🐳 Docker containerization
- 🔄 MLOps pipeline with MLflow
- ✅ 95%+ test coverage
- 📊 Data visualization tools

## ⚡ Quick Start

```bash
git clone https://github.com/Muhammad-Farooq-13/OPencv.git
cd OPencv
pip install -r requirements.txt
python flask_app.py
```

Visit: http://localhost:5000
